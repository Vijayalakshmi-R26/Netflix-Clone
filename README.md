# Netflix Clone

This is my first frontend project where I recreated the Netflix landing page using HTML and CSS.

## Project Description

This project is a static clone of the Netflix homepage designed to improve my frontend development skills. 
I focused on layout design, responsive structure, and styling similar to the original Netflix website.

## Features

- Netflix-style homepage layout
- Hero banner section
- Movie thumbnail sections
- Responsive design
- Clean UI using Flexbox and CSS styling

## Technologies Used

- HTML5
- CSS3

This project helped me understand webpage structuring, CSS positioning, flexbox, and responsive design. 
It improved my confidence in building real-world UI layouts.
## Movie Section

Includes a custom section displaying my favourite movies with posters and descriptions.
